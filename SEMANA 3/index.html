Bienvenido a mi Proyecto Web

Sobre el Proyecto

Este proyecto corresponde a la Semana 3 de la materia de Desarrollo Web. Aquí aplicamos conocimientos básicos de HTML5.

Objetivos de Aprendizaje

El objetivo es comprender la estructura básica de una página web, utilizar correctamente los encabezados y aplicar formatos de texto como negrita y cursiva.
